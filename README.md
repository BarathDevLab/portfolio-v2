# Barath G — Developer Portfolio v2

Personal developer portfolio showcasing projects, skills, and experience as a Full Stack Developer.

## Live Demo
https://portfolio-v2-tawny-kappa.vercel.app

## Tech Stack
- **Framework:** Next.js, TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Features
- Projects showcase with live demo links
- Skills and tech stack overview
- Internship and experience timeline
- Responsive design
- Contact links

## Getting Started
```bash
git clone https://github.com/BarathDevLab/portfolio-v2
cd portfolio-v2
npm install
npm run dev
```
